# Portfolio Website

Welcome to my personal portfolio website, built with Framer. This portfolio showcases my work, skills, and experience in a clean, modern design.

## Features

- Responsive design that works on all devices
- Modern UI with smooth animations
- Easy to customize and update
- Optimized for performance

## Getting Started

1. Clone this repository
2. Open `page.html` in your browser to view the portfolio

## Deployment

This portfolio can be deployed to any static hosting service like GitHub Pages, Netlify, or Vercel.

## Technologies Used

- HTML5
- CSS3
- JavaScript
- [Framer](https://www.framer.com/)
